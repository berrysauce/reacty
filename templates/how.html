How does reacty work?

STEP 1 
First, sign up via the big "Sign up" button at the top. Just enter your domain (with http:// or https://) and choose a password for your dashboard.

STEP 2
Copy the widget code from your dashboard and add it inside the <head></head> section of your website. The widget will be shown on every page the code was added to. If you're using a website builder, search for the setting to add custom code to your header.

STEP 3
If you've done everything correctly, the widget should pop up on your website. Submitted feedback will be visible in your dashboard.

It's as easy as that. Got more questions? Go ahead, ask them!
